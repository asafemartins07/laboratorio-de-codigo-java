# 📚 Projetos em Java — Estudos de Programação Orientada a Objetos

Este repositório reúne pequenos projetos desenvolvidos durante estudos em **Java**, com foco em conceitos fundamentais da linguagem e introdução à **Programação Orientada a Objetos (POO)**.

---

## 🚀 Conteúdo do Repositório

* Sintaxe básica em Java
* Tipos de dados primitivos e objetos
* Entrada e saída de dados
* Estruturas condicionais (`switch`)
* Estruturas de repetição (`for`)
* Manipulação de Strings
* Organização de código

---

## 🎯 Objetivo

Este repositório tem como objetivo consolidar conhecimentos em Java e servir como base para evolução em conceitos mais avançados, especialmente em **Programação Orientada a Objetos**, que será aprofundada em projetos futuros.

---

## 🛠️ Tecnologias Utilizadas

* Java
* Terminal / Console
* IDE (VS Code)

---

## 📌 Observações

Os códigos são de caráter educacional e podem ser aprimorados conforme o avanço nos estudos, incluindo:

* Melhor tratamento de erros
* Validação de entrada do usuário
* Aplicação mais estruturada de POO (classes, métodos, encapsulamento)

---

Feito para aprendizado, prática e evolução constante 🚀
